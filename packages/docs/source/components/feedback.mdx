## Installation

```bash
npx install-peerdeps @availity/feedback --save
```

## Usage

```jsx live=true
import Feedback from '@availity/feedback';

<div className="w-100 d-flex flex-column justify-content-around align-items-start">
  <Feedback
    appName="Payer Space"
    prompt="Please provide some feedback"
    color="primary"
  >
    Provide Feedback
  </Feedback>
</div>;
```

## Feedback (Default export)

Availity Feedback component

### Props

| Prop Name            | Types    | Required | Default                                      | Description                                                                                                               |
| -------------------- | -------- | -------- | -------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| **`appName`**        | String   | false    |                                              | The name of the application this feedback is for. It is used in the API request to indicate where the feedback came from. |
| **`formProps`**      | Object   | false    |                                              | Props to be spread to the underlying `FeedbackForm`. See the props for `FeedbackForm` below.                              |
| **`children`**       | Node     | false    | `"Give Feedback"`                            | The text to display on the submit button.                                                                                 |
| **`className`**      | String   | false    |                                              | Additional styles to be displayed on the dropdown                                                                         |
| **`outline`**        | Boolean  | false    |                                              | Indicates if the button should use the "outline" styles or not.                                                           |
| **`color`**          | String   | false    | `"light"`                                    | The color of the button.                                                                                                  |
| **`prompt`**         | String   | false    | `"Tell us what you think about ${appName}."` | Text that prompts the user to provide feedback.                                                                           |
| **`onFeedbackSent`** | Function | false    |                                              | Callback for when the feedback is submitted. It is called with the feedback object.                                       |
| **`modal`**          | Boolean  | false    | `false`                                      | Whether to open the `FeedbackForm` in a modal                                                                             |

## FeedbackForm

This is the underlying form, which is exposed in case you need to gather feedback inline.

### Props

| Prop Name                | Types           | Required | Default                                   | Description                                                                                                                                                                                                                           |
| ------------------------ | --------------- | -------- | ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`name`**               | String          | true     |                                           | The name of the application this feedback is for. It is used in the API request to indicate where the feedback came from.                                                                                                             |
| **`onFeedbackSent`**     | Function        | false    |                                           | Callback for when the feedback is submitted. It is called with the feedback object.                                                                                                                                                   |
| **`faceOptions`**        | Array<`Object`> | false    |                                           | Array of Objects containing **`icon`** (String), **`description`** (String), and **`placeholder`** (String) properties. Allows you to override the smiley face options which appear. Default: Smiley Face, Meh Face, and Frowny Face. |
| **`aboutOptions`**       | Array<`Object`> | false    |                                           | Array of Objects containing **`value`** (String,Number) and **`label`** (String) properties. Allows a dropdown displaying the options provided to let the user indicate what the feedback is about.                                   |
| **`onClose`**            | Function        | false    |                                           | When provided, a "Close" button is rendered and `onClose` is executed when it is clicked                                                                                                                                              |
| **`prompt`**             | String          | false    | `"Tell us what you think about ${name}."` | Text that prompts the user to provide feedback.                                                                                                                                                                                       |
| **`additionalComments`** | Boolean         | false    | `false`                                   | If `true`, shows an optional comments field below.                                                                                                                                                                                    |
| **`staticFields`**       | Object          | false    |                                           | Static (non-user-entered) key/value pairs to be sent in feedback submission                                                                                                                                                           |

### Usage

```jsx live=true
import { FeedbackForm } from '@availity/feedback';

<div className="w-100">
  <FeedbackForm name="Payer Space" />
</div>;
```
