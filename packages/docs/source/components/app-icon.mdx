A component which outputs one of the "App Icons" from [Availity UI Kit](http://availity.github.io/availity-uikit/v3/components#App-Icons)

## Installation

```bash
npm install @availity/app-icon --save
```

## Usage

import AppIcon from '@availity/app-icon';

<div className="w-100 d-flex flex-row justify-content-around align-items-center">
<AppIcon title="Payer Space" color="blue" branded size="xl">
  PS
</AppIcon>
<AppIcon title="Payer Space" color="red" size="xl">
  PS
</AppIcon>
<AppIcon title="Payer Space" color="orange">
  PS
</AppIcon>
</div>

```jsx
import AppIcon from '@availity/app-icon';

<AppIcon title="Payer Space" color="green" branded size="xl">
  PS
</AppIcon>
```

## Props

| Prop Name     | Types   | Required | Default   | Description                                                                            |
| ------------- | ------- | -------- | --------  | -------------------------------------------------------------------------------------- |
| **`size`**    | String  | false    |           | Potential values: `"lg"`, `"xl"`                                                       |
| **`color`**   | String  | false    | `"black"` | Potential values: `"black"`, `"blue"`, `"green"`, `"orange"`, `"red"`                  |
| **`branded`** | Boolean | false    |           | Triggers "branded" styles.                                                             |
| **`src`**     | String  | false    |           | If image source is provided will render this instead of children.                      |
| **`alt`**     | String  | false    |           | for `src` prop. the alt property for your image source                                 |
