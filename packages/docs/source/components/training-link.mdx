title: Training Link
summary: Component for allowing link out to training in the Header component
---

[![Version](https://img.shields.io/npm/v/@availity/training-link.svg?style=for-the-badge)](https://www.npmjs.com/package/@availity/training-link)

Provide a link for training videos for your app

## Installation

```bash
npm install @availity/training-link --save
```

## Usage

```jsx
import React from 'react';
import TrainingLink from '@availity/training-link';

<TrainingLink link={link} name={name}>
  {/* ... */}
</TrainingLink>
```

## Props

| Prop Name     | Types   | Required | Default  | Description                                                                            |
| ------------- | ------- | -------- | -------- | -------------------------------------------------------------------------------------- |
| **`link`**    | String  | true     |          | Link to training video                                                                 |
| **`name`**    | String  | true     |          | Name of your app to make the sentence complete                                         |
