## Installation

```bash
npx install-peerdeps @availity/favorites --save
```

## Usage

import Favorites from './Favorites';
import '@availity/favorites/style.scss';

<div className="w-100 d-flex flex-row justify-content-around align-items-center">
  <Favorites>
    {({ default: FavoritesProvider, FavoriteHeart }) => (
      <FavoritesProvider>
        <FavoriteHeart id="123456" />
      </FavoritesProvider>
    )}
  </Favorites>
</div>

```jsx
import React from 'react';
import Favorites, { FavoriteHeart } from '@availity/favorites';
import '@availity/favorites/style.scss';

<Favorites>
  <FavoriteHeart id="123456" />
</Favorites>;
```

## Favorites

The Provider Component that stores all of the user's favorites for use in the favorite components.

### Props

| Prop Name      | Types | Required | Default | Description                                     |
| -------------- | ----- | -------- | ------- | ----------------------------------------------- |
| **`children`** | Node  | false    |         | Children to render inside of Favorites provider |

## FavoriteHeart

The default Favorite Icon that can be used to store a favorited item. Can only be used inside of a `<Favorites />` Provider.

### Props

| Prop Name      | Types    | Required | Default | Description                                                   |
| -------------- | -------- | -------- | ------- | ------------------------------------------------------------- |
| **`id`**       | String   | true     |         | The unique id of the favorite item. Will be fetched from API. |
| **`onChange`** | Function | false    |         | Called once the favorite heart has been clicked and updated.  |

## useFavorites

Hook that will allow you to "hook" into the favorites logic and create a custom component with the same logic. Can only be used inside of a `<Favorites />` Provider.

### Arguments

- **`id`**: The id of the favorite you want to create/update.

### useFavorites Usage

```jsx
import React, { useState } from 'react';
import { useFavorites } from '@availity/favorites';

const Component = () => {
  const [isFavorite, toggleFavorite] = useFavorites('1234');

  return (
    <Icon
      name={isFavorite ? 'plus-circle' : 'minus-circle'}
      onClick={toggleFavorite}
    />
  );
};
```
