[![Version](https://img.shields.io/npm/v/@availity/page-header.svg?style=for-the-badge)](https://www.npmjs.com/package/@availity/page-header)

The standard Availity application header that appears at the top of the page. It can include an app icon, payer logo, and/or the feedback loop.

## Installation

```bash
npx install-peerdeps @availity/page-header --save
```

## Usage

```jsx live=true
import PageHeader from '@availity/page-header';

<div className="w-100 d-flex flex-column justify-content-around align-items-start">
  <PageHeader appName="Payer Space" appAbbr="PS" iconColor="blue" feedback />
</div>;
```

## Props

| Prop Name           | Types                | Required | Default      | Description                                                                                                                                                                                                                                   |
| ------------------- | -------------------- | -------- | ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`appName`**       | String               | true     |              | Name of the application                                                                                                                                                                                                                       |
| **`spaceName`**     | String               | false    |              | Payer Space Name of the space in which this page is under.                                                                                                                                                                                    |
| **`spaceId`**       | String               | false    | `null`       | Payer Space ID of the space in which this page is under.                                                                                                                                                                                      |
| **`appAbbr`**       | String               | false    |              | If provided and payerId is not provided, the app icon will appear.                                                                                                                                                                            |
| **`iconColor`**     | String               | false    | `"black"`    | Potential values: `"black"`, `"blue"`, `"green"`, `"orange"`, `"red"`. Only used if the app icon should appear.                                                                                                                               |
| **`branded`**       | Boolean              | false    |              | Triggers the app icon's "branded" styles. Only used if the app icon should appear.                                                                                                                                                            |
| **`payerId`**       | String               | false    | `null`       | The ID of the payer the application is for. If provided, the payer logo appears and not the app icon.                                                                                                                                         |
| **`component`**     | React Element        | false    |              | Allow rendering of an optional component in the top right of the header.                                                                                                                                                                      |
| **`feedback`**      | Boolean              | false    | `false`      | If `true`, the feedback loop button appears.                                                                                                                                                                                                  |
| **`feedbackProps`** | Boolean              | false    |              | See [Feedback](https://github.com/Availity/availity-react/tree/master/packages/feedback). Props to send to `<Feedback />` component                                                                                                           |
| **`children`**      | Node                 | false    | `${appName}` | This value appears at the `h1` of the page. Overrides the `appName` to allow for custom elements (such as a "beta" tag)                                                                                                                       |
| **`crumbs`**        | Node,Array<`Object`> | false    |              | Array of Objects containing `name` (String) and `url` (String) properties. The ancestor pages which get passed to the `Breadcrumbs` component. See [Breadcrumbs](https://github.com/Availity/availity-react/tree/master/packages/breadcrumbs) |
| **`tag`**           | Function,String      | false    | `<h1>`       | The tag to render the page header as                                                                                                                                                                                                          |
| **`clientId`**      | String               | false    |              | Client ID to use in [Spaces](https://github.com/Availity/availity-react/tree/master/packages/spaces) to fetch the payer's logo                                                                                                                |
| **`iconSrc`**       | String               | false    |              | Image source for `<AppIcon />` to be used instead of `appAbbr`                                                                                                                                                                                |
| **`iconAlt`**       | String               | false    |              | Required if `iconSrc` is used. Image alt property of `<AppIcon />`                                                                                                                                                                            |
