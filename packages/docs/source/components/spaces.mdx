## Spaces (Default export)

This is the provider component needed for `@availity/spaces` components to work. All `@availity/spaces` components must be children of a Spaces provider.

### Props

| Prop Name       | Types           | Required | Default  | Description                                                                                                                                                                  |
| -------------   | -------         | -------- | -------- | --------------------------------------------------------------------------------------                                                                                       |
| **`clientId`**  | String          | true     |          | The Client ID obtained from APIConnect. Must be subscribed to the slotmachine API.                                                                                           |
| **`query`**     | String          | false    |          | Override the default slotmachine query                                                                                                                                       |
| **`variables`** | Object          | false    |          | Override the default variables used in the slotmachine query                                                                                                                 |
| **`spaces`**    | Array<`Object`> | false    |          | Array of spaces to be passed into the Spaces provider. Useful for if you already have the spaces in your app and don't want the spaces provider to have to fetch them again. |
| **`spaceIds`**  | Array<`String`> | false    |          | Array of spaceIds the Spaces provider should fetch the spaces for. Any `spaceIds` already included in `spaces` will not be fetched again.                                    |
| **`payerIds`**  | Array<`String`> | false    |          | Array of payerIds the Spaces provider should fetch the spaces for. Any `payerIds` already included in `spaces` will not be fetched again.                                    |

## Images

### Usage

```jsx
import React from 'react';
import Spaces, {
  SpacesLogo,
  SpacesBillboard,
  SpacesTile,
} from '@availity/spaces';

<Spaces
  spaceIds={['73162546201441126239486200007187']}
  payerIds={['PayerID']}
  clientId="my-client-id"
>
  <SpacesLogo spaceId="73162546201441126239486200007187" />
  <SpacesBillboard payerId="PayerID" />
  <SpacesTile payerId="PayerID" />
</Spaces>;
```

### SpacesLogo (Named export)

Display the Space's logo

### SpacesTile (Named export)

Display the Space's tile

### SpacesBillboard (Named export)

Display the Space's billboard

### Props

| Prop Name           | Types   | Required | Default  | Description                                                                                                                                                                                                                                                                               |
| -------------       | ------- | -------- | -------- | --------------------------------------------------------------------------------------                                                                                                                                                                                                    |
| **`spaceId`**       | String  | false    |          | Required if `payerId` is not provided. The payer spaces ID of the payer to render the image for. If no `spaceId` or `payerId` is provided, the first space in the `spaces` array is used. Note: This is only to be used when the Spaces provider should only ever contain a single space. |
| **`payerId`**       | String  | false    |          | Required if `spaceId` is not provided. The payer ID of the payer to render the image for. If no `spaceId` or `payerId` is provided, the first space in the `spaces` array is used. Note: This is only to be used when the Spaces provider should only ever contain a single space.        |
| **`skeletonProps`** | Object  | false    |          | Dimensions passed to loader to show while the image is loading.                                                                                                                                                                                                                           |
| **`fallback`**      | String  | false    |          | The fallback image url to render if the url for the spaces image is not valid or not found                                                                                                                                                                                                |

## SpacesDisclaimer (Named Export)

Display the disclaimer for a given space id.

### Props

| Prop Name      | Types   | Required | Default  | Description                                                                                                                                                                                                                   |
| -------------  | ------- | -------- | -------- | --------------------------------------------------------------------------------------                                                                                                                                        |
| **`spaceId`**  | String  | false    |          | The id of the space to render the disclaimer for. If no `spaceId` is provided, the first space in the `spaces` array is used. Note: This is only to be used when the Spaces provider should only ever contain a single space. |
| **`markdown`** | Boolean | false    | `false`  | Render the disclaimer as markdown                                                                                                                                                                                             |
| **`styled`**   | Boolean | false    |          | See [Disclaimer](https://github.com/Availity/availity-react/tree/master/packages/typography)                                                                                                                                  |

### Usage

```jsx
import React from 'react';
import Spaces, { SpacesDisclaimer } from '@availity/spaces';

<Spaces spaceIds={['73162546201441126239486200007187']} clientId="my-client-id">
  <SpacesDisclaimer
    markdown
    styled
  />
</Spaces>;
```

## SpacesGhostText (Named Export)

Display the ghost text for a given space. Renders `null` if the ghost text should not display.

### Props

| Prop Name     | Types   | Required | Default  | Description                                                                                                                                                                                                                   |
| ------------- | ------- | -------- | -------- | --------------------------------------------------------------------------------------                                                                                                                                        |
| **`spaceId`** | String  | false    |          | The id of the space to render the ghost text for. If no `spaceId` is provided, the first space in the `spaces` array is used. Note: This is only to be used when the Spaces provider should only ever contain a single space. |

### Usage

```jsx
import React from 'react';
import Spaces, { SpacesGhostText } from '@availity/spaces';

<Spaces spaceIds={['73162546201441126239486200007187']} clientId="my-client-id">
  <SpacesGhostText />
</Spaces>;
```

## useSpace

This is a custom hook for grabbing any Spaces data you may need from the `Spaces` provider.

### Usage

```jsx
import React from 'react';
import { useSpace } from '@availity/spaces';

const SpacesComponent = () => {
  // id can be a space or a payer id
  const { space, isGhost, loading } = useSpace(id);
};
```

## SpacesContext

If you are using a class component, you can subscribe to the spaces by using this context.

### Usage

```jsx
import React from 'react';
import { SpacesContext } from '@availity/spaces';
// ...
class SpacesComponent extends React.Component {
  render() {
    // id can be a space or a payer id
    const { space, images } = this.context;
    // Returns space and images for id
  }
}

SpacesComponent.contextType = SpacesContext;
```
