[![Version](https://img.shields.io/npm/v/@availity/avatar.svg?style=for-the-badge)](https://www.npmjs.com/package/@availity/avatar)

## Installation

```bash
npm install @availity/avatar --save
```

## Usage

```javascript
import React from 'react';
import Avatar from '@availity/avatar';
// ...
<Container>
  <Avatar />
</Container>
// ...
```

## Props

| Prop Name           | Types   | Required | Default                                           | Description                                                                            |
| -------------       | ------- | -------- | --------                                          | -------------------------------------------------------------------------------------- |
| **`skeletonProps`** | Object  | false    |                                                   | Dimensions passed to loader to show while the avatar is loading.                       |
| **`fallback`**      | String  | false    | `"/public/apps/my-profile/images/Avatars-00.png"` | Image url to render if the url for the avatar is not not found.                        |
