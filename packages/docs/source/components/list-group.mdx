Availity's ListGroup, which can feature Cards and Selectable items

## Installation

```bash
npm install @availity/list-group --save
```

## Usage

```jsx live=true
import ListGroup from '@availity/list-group';
import ListGroupItem from '@availity/list-group-item';

<div className="w-100 d-flex flex-row justify-content-around align-items-center">
  <ListGroup selectable cards>
    <ListGroupItem>Item</ListGroupItem>
    <ListGroupItem>Item</ListGroupItem>
    <ListGroupItem>Item</ListGroupItem>
  </ListGroup>
</div>
```

## Props

| Prop Name        | Types           | Required | Default  | Description                                                                            |
| -------------    | -------         | -------- | -------- | -------------------------------------------------------------------------------------- |
| **`cards`**      | Boolean         | false    |          | Triggers the items (children) to appear as cards                                       |
| **`selectable`** | Boolean         | false    |          | Triggers the items (children) to appear selectable when hovered over                   |
| **`tag`**        | Function,String | false    | `<ul>`   | The tag to use in the list group                                                       |
