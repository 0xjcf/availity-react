### Installation

npm

```bash
npm install @availity/loading-button --save
```

Yarn

```bash
yarn add @availity/loading-button
```

### Example

```jsx
import React from 'react';
import Progress from '@availity/loading-button';

const Example = () => (
  <>
     <LoadingButton block disabled isLoading={isLoading} color="primary">
            Test
      </LoadingButton>
  </>
);
```

#### Live example: <a href="https://availity.github.io/availity-react/storybook/?path=/story/components-loading-button--default"> Storybook</a>

### Props

#### `isLoading: boolean`

This is the flag that determines whether the button is loading. If it is true, the loading panel will show.

#### `className?: string`

Additional classes that should be applied to the button.

#### `type?: string`

This represents the button type, so either 'submit' or 'button'. Defaults to 'submit'.
